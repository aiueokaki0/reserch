Inside the Archive : 
- All the source and run files, contained within the "TeamFormation" folder

- 3 demos in the .webm format :
-- demoTonyParis is a demo of the original software that was used in the Team Formation paper (in the Papers directory)
-- demoNoCom is a demo of the software i've made using communications between agents
-- demo2 is a demo of the software i've made without using communications between agents

- 3 scripts in the .sh format (that just need to be run using a shell terminal) :
-- MTF_Kobe is a scenario in the town of Kobe
-- MTF_Paris is a scenario in the town of Paris
-- MTF_Loop loops both scenarios one after the other
